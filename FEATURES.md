# Vently - Features Overview

## üé§ Audio Features

### Recording
- **Browser-based Recording**: Record audio directly in your browser using the Web Audio API
- **Real-time Timer**: See recording duration as you record
- **One-click Recording**: Simple start/stop controls
- **Audio Format**: Records in WebM format, compatible with all modern browsers

### Upload
- **File Upload**: Upload pre-recorded audio files
- **Supported Formats**: MP3, WAV, OGG, M4A, WebM, AAC
- **File Size Limit**: Up to 10MB per audio file
- **Drag & Drop**: Easy file upload interface

### Playback
- **Custom Audio Player**: Beautiful, intuitive audio player
- **Play/Pause Controls**: Easy playback controls
- **Progress Bar**: Visual representation of playback progress with seek functionality
- **Time Display**: Shows current time and total duration
- **Volume Control**: Adjustable volume with mute option
- **Responsive Design**: Works seamlessly on all devices

## üë• Social Features

### User Profiles
- **Personalized Profiles**: Each user has a unique profile page
- **Display Name**: Set a display name separate from username
- **Bio**: Add a short bio about yourself (up to 200 characters)
- **Profile Avatar**: Visual representation with username initial or custom image
- **User Stats**: View follower count, following count, and post count
- **User Posts**: See all posts from a specific user

### Following System
- **Follow/Unfollow**: Build your community by following other users
- **Followers List**: See who follows you
- **Following List**: See who you follow
- **Follow Button**: Easy one-click follow/unfollow on profiles

### Feed
- **Home Feed**: See posts from users you follow and your own posts
- **Discover Feed**: Explore posts from the entire community
- **Chronological Order**: Posts sorted by creation time (newest first)
- **Infinite Scroll**: Load more posts as you scroll
- **Real-time Updates**: See new posts as they're created

- **Post Captions**: Add context to your audio with text captions
